# Contact-Management-System
Features:
✔ Add new contact
✔ Search by name
✔ Display all contacts
✔ Dictionary-based fast lookup
✔ Functions + clean structure
✔ Beginner-friendly code

Description:
The Contact Management System is a Python-based program that stores and manages contact information using a dictionary. It allows users to add new contacts, search for existing ones, and display all saved contacts through simple menu-driven options. The project demonstrates the use of functions, loops, and dictionary operations, making it a useful beginner-level Python application.
